tus-node-client
===============

[TUS Protocol 0.2.1](http://www.tus.io/protocols/resumable-upload.html) Node Client Implementation

## Installation
```
npm install
```

## Example
```
./tus-node-client.js macaca.jpg http://localhost:1080/files/
```
