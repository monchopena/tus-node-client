tus-node-client
===============

NodeJS client for the tus (http://tus.io) protocol.

First:

<pre>
npm install
</pre>

Sample:

<pre>
./tus-node-client.js macaca.jpg http://localhost:1080/files/
</pre>

If you need a Tus Node Server you can test Brewtus (https://github.com/vayam/brewtus).

Have fun!
